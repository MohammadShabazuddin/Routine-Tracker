Install RoutineFlow

Android steps
  1. Open this link in Chrome on Android.
  2. Tap the Install button or use the ⋮ menu → Install app.
  3. RoutineFlow will appear on the Home Screen like a normal app.

Tip: If you don’t see the prompt, refresh the page and try again.

Reminders Not Showing?

Quick fix
  1. Open the app in Chrome on Android.
  2. Tap Install and open the app from the Home Screen.
  3. Go to Settings and enable reminders.
  4. When prompted, allow notifications.

Device Settings

Android
  • Settings → Apps → Chrome → Notifications → Allow
  • Settings → Apps → RoutineFlow → Notifications → Allow
  • Disable battery optimization for Chrome or RoutineFlow

How It Works

Good to know

Reminders always work when the app is open. Background reminders are best effort on Android PWAs and can be throttled by battery saver.
